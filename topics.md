## Adding a New Topic

1. Create a new shared-understanding repo:
   ```bash
   mkdir ~/git/topic-name
   cd ~/git/topic-name
   git init
   # Copy structure from existing topic repo
   ```

2. Add to this table with status "Active"

3. Create a summary in `summaries/topic-name.md`

4. Add relevant channels to `channels.md`
