# People

Team member profiles for context and collaboration.

## Structure

Each person gets a directory with:
```
people/
└── username/
    ├── profile.md      # Role, expertise, contact
    └── notes.md        # Working notes, preferences
```

## Adding a Person

```bash
mkdir people/username
```

Create `profile.md`:
```markdown
# Display Name

**Slack:** @username
**Role:** Job Title
**Team:** Team Name

## Expertise
- Area 1
- Area 2

## Notes
Working preferences, context, etc.
```
