# Summaries

Living summaries for each topic. Updated as understanding evolves.

## Format

Each topic gets a summary file: `topic-name.md`

```markdown
# Topic Name Summary

**Last Updated:** YYYY-MM-DD

## Overview
High-level description of the topic.

## Key Concepts
- Concept 1
- Concept 2

## Current Status
What's happening now.

## Open Questions
- Question 1
- Question 2

## Related Topics
- [Related Topic](related-topic.md)
```
